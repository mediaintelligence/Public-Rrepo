## Installation Steps

### Step 1: Download & Extract

1. Download `chrome-boss-agent-extension-fixed.zip`
2. Extract the ZIP file:
   - **Mac:** Double-click the zip file
   - **Windows:** Right-click → "Extract All..."

**Important:** After extracting, you'll have a nested folder structure:
```
chrome-boss-agent-extension/
└── chrome-boss-agent-extension/   ← This is the folder you need
    ├── manifest.json
    ├── icons/
    └── src/
```

---

### Step 2: Open Chrome Extensions

1. Open **Google Chrome**
2. Type `chrome://extensions/` in the address bar
3. Press **Enter**

---

### Step 3: Enable Developer Mode

1. Find the **"Developer mode"** toggle in the top-right corner
2. Turn it **ON** (toggle turns blue)

---

### Step 4: Load the Extension

1. Click **"Load unpacked"** button (top-left)
2. Navigate to the extracted folder
3. **Select the INNER `chrome-boss-agent-extension` folder** (the one containing `manifest.json`)
4. Click **"Select"** (Mac) or **"Select Folder"** (Windows)

---

### Step 5: Verify Installation

You should see:
- "MIZ OKI Boss Agent" card on the extensions page
- Version: 1.0.0
- Blue toggle (enabled)

**If you see errors:** Make sure you selected the correct folder (the inner one with `manifest.json`).

---

### Step 6: Pin to Toolbar

1. Click the puzzle piece icon in Chrome toolbar
2. Find "MIZ OKI Boss Agent"
3. Click the pin icon to keep it visible

## Troubleshooting

### "Service worker registration failed"
- Make sure you selected the correct folder (inner folder with `manifest.json`)
- Try removing and re-loading the extension

### Extension icon not visible
1. Click puzzle piece icon in toolbar
2. Pin "MIZ OKI Boss Agent"

### "Manifest file is invalid"
- You may have selected the wrong folder
- Select the folder that directly contains `manifest.json`

### Side panel won't open
- Update Chrome to version 114 or higher
- Restart Chrome after updating

### API connection errors
- Check your internet connection
- The Boss Agent server may be temporarily unavailable

## Updating the Extension

When a new version is available:

1. Download and extract the new ZIP
2. Go to `chrome://extensions/`
3. Click the refresh icon on the "MIZ OKI Boss Agent" card
4. Or remove and re-load the extension
